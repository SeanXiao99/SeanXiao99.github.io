报： to set your account's default identity. Omit --global to set the identity only in this repository.

fatal: unable to auto-detect email address (got 'root@ubuntu.(none)')

解决办法： 执行下面俩步骤： git config --global user.email "you@example.com"

git config --global user.name "Your Name"
